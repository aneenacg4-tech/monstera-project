## Overview

The Monstera landing page is a visually engaging website designed to highlight Monstera plants. It is responsive for mobile, tablet, and desktop screens and includes a navigation bar, hero section, and informational sections on plant care.

---

## Features

- Responsive design for all screen sizes
- Hero section with large, bold typography
- Navigation menu with hover effects
- Interactive button with icon
- Informational section on plant care (Feeding, Light, Care)
- Uses Tailwind CSS for styling and Font Awesome for icons

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/monstera-landing-page.git
````

2. Navigate into the project directory:

   ```bash
   cd monstera-landing-page
   ```

3. Open `index.html` in your browser:

   ```bash
   open index.html
   ```

## Usage

* Click navigation items to explore different sections (placeholder links currently).
* On desktop, a large hero image appears on the right; on mobile, a smaller optimized image is shown.
* Hover over navigation items for underline effects.
* Scroll down to view plant care information.
